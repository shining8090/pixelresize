LinkedIn Image Size Guide

For the best professional appearance, use these dimensions:

  • Personal Profile Picture: 400 x 400 pixels
  • Background Photo (Banner): 1584 x 396 pixels
  • Company Logo: 300 x 300 pixels
  • Shared Image (Post): 1200 x 627 pixels
