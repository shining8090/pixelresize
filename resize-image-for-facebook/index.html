Facebook Image Size Guide

To avoid automatic cropping or blurriness, use these standard Facebook sizes:

  • Profile Picture: 170 x 170 pixels
  • Cover Photo: 820 x 312 pixels (Desktop)
  • Feed Post: 1200 x 630 pixels
  • Story: 1080 x 1920 pixels
